# 4CornersEatery 🍽️

A modern, luxury restaurant website built using **HTML**, **Tailwind CSS**, and **JavaScript**.  
This project focuses on a premium UI design with a responsive navigation bar and a hero section featuring a blurred background image.

---

## ✨ Features

- Modern luxury restaurant UI design
- Fully responsive layout (mobile-first)
- Sticky navbar with glassmorphism effect
- Hamburger menu for mobile navigation
- Hero section with:
  - Blurred background image
  - Dark overlay for readability
  - Call-to-action buttons
- Smooth hover effects and transitions
- Clean and minimal design aesthetic

---

## 🛠️ Tech Stack

- **HTML5** – Structure
- **Tailwind CSS** – Styling (compiled, no CDN)
- **JavaScript (Vanilla)** – Hamburger menu functionality
