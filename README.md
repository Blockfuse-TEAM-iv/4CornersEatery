# 4Corners Eatery

4Corners Eatery is a premium restaurant website featuring authentic Nigerian and continental cuisine. The project focuses on modern luxury aesthetics, providing a high-end visual experience for diners.

---

## Core Features

- **Premium Landing Page**: A sophisticated hero section with dark luxury aesthetics, featuring blurred background elements and circular food previews.
- **Categorized Menu Display**: A dedicated menu page where dishes are organized into logical culinary categories (Soups, Rice, Grilled, Drinks, and Desserts).
- **Responsive Navigation**: A sticky navbar with glassmorphism effects and a mobile-responsive hamburger menu for seamless browsing across devices.
- **Navigation Utilities**: Implementation of smooth scrolling and a "Back to Top" button for enhanced site usability.
- **Detailed Footer**: Comprehensive business details including location information, contact channels, and weekly opening hours.

---

## Technical Stack

- **HTML5**: Structured semantic markup for accessibility and SEO.
- **Tailwind CSS**: Utility-first styling with a custom production build for optimal performance.
- **Vanilla JavaScript**: Powers the mobile hamburger menu toggle and handles basic navigation state management.

---

## Project Structure

- `index.html`: The primary landing page showcasing the brand identity and hero content.
- `menu.html`: A detailed menu page organized by culinary category.
- `script.js`: Core client-side logic focusing on navigation and responsive interactivity.
- `src/output.css`: The compiled production-ready stylesheet.
- `images/`: Local repository for high-resolution food and brand assets.
